EssayStack

A simple bookmark tracker to save interesting essays and websites. Sorted by read status, priority, and date.

Note: For security reasons, write access for adding, modifying, and deleting essays has been disabled on the public website. To host this project on your own, clone the repo, use your own Firebase database, and update firebaseConfig as necessary.

For now, enjoy the author's recommentations below. Happy reading!
